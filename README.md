# 🚀 GitHub Performance Finder

A simple web application built using **HTML, CSS, and JavaScript** that allows users to search GitHub profiles and analyze their performance based on repositories, followers, and activity.

---

## 🌐 Live Demo
👉 https://github-performance-finder-7j6l.vercel.app/

---

## 📌 Features
- 🔍 Search any GitHub username
- 📊 View profile details (followers, following, repos)
- ⭐ Highlight top repositories
- ⚡ Fast and responsive UI
- ❌ Error handling for invalid usernames

---

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- GitHub API

---

## 📁 Project Structure
/project-folder
│── index.html
│── github.css
│── github.js


---

## 🚀 How to Use
1. Enter a GitHub username  
2. Click search button  
3. View user details instantly  

---

## ⚙️ Deployment
This project is deployed using **Vercel**.

To deploy:
1. Upload project folder  
2. Select **Other (Static Site)**  
3. Click deploy
